# midex

Multi-agent (persona) workflow orchestration platform.

Modular, extensible, global-first approach. Customize for a single developer, or adapt for team use.


## Use Case: Lite | Standard
- **Lite:** No runtime, no overhead. Plug-n-play to immediately use rules, agent personas and workflows.
- **Standard:** Full-spec runtime. Inter-session, cross-project awareness for truly lossless context. Retain and learn from experiences. Web interface with realtime telemetry.
